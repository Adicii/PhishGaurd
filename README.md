# PhishGuard — Context-Aware Phishing Detection

ML-based phishing detection system combining structural URL analysis with behavioral session signals.

## Architecture

- **Base models**: XGBoost, LightGBM, Random Forest (5-fold CV, isotonic calibration)
- **Meta-learner**: Logistic Regression (stacking ensemble)
- **Fusion**: 65% URL score + 35% behavioral score (late fusion)
- **Threshold**: Adaptive with ±0.08 uncertainty zone

## Performance

| Metric | Score |
|--------|-------|
| F1 Score | 0.9806 |
| ROC-AUC | 0.9967 |
| Accuracy | 0.9996 |

## Setup

```bash
pip install -r requirements.txt
```

## Train models

```bash
python src/train.py
```

Model files are saved to `models/` (not committed to Git due to size).

## Run dashboard

```bash
python -m streamlit run dashboard.py
```

## Project structure

```
src/
  feature_extractor.py   # 21 URL features
  behavior_model.py      # Behavioral session scoring
  pipeline.py            # Unified PhishGuardPipeline class
  threshold.py           # Adaptive threshold computation
  train.py               # Model training script
models/                  # Generated by train.py (gitignored)
data/                    # Training datasets (gitignored)
results/                 # CV results and figures (gitignored)
dashboard.py             # Streamlit dashboard
```

## Known limitations

- External validation accuracy ~50.5% due to dataset distribution shift
- Behavioral fusion is score-level (late fusion), not feature-level
- NLP tokenization signals displayed but not fed into stacking ensemble
- No live WHOIS or HTTP fetching
